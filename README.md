# MMA-NET: Dual Transformer Fusion with Cross-Modal Attention for Early Parkinson Detection

## Overview
Parkinson’s disease is a progressive neurological disorder that requires early diagnosis for effective treatment.  
This project proposes **MMA-NET**, a deep learning framework that integrates **Dual Transformer Fusion** with a **Cross-Modal Attention Mechanism** for robust early detection of Parkinson’s disease using multimodal data.

The model learns complex relationships between heterogeneous data sources and improves prediction accuracy.

---

## Key Features
- Dual Transformer architecture for feature extraction
- Cross-modal attention mechanism
- Multimodal data fusion
- Early detection of Parkinson’s disease
- Robust classification performance

---

## Architecture
The MMA-NET framework consists of the following components:

1. Multimodal Feature Extraction  
2. Dual Transformer Encoders  
3. Cross-Modal Attention Fusion  
4. Classification Layer  

The architecture enables the model to capture complex interactions between different modalities.
