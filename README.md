# MMA-NET: Dual Transformer Fusion with Cross Modal Attention for Early Parkinson Detection

## Overview

This project proposes a deep learning architecture called MMA-NET for early detection of Parkinson's disease using multimodal data.

## Features

- Dual Transformer Architecture
- Cross Modal Attention Mechanism
- Multimodal Data Fusion
- Early Parkinson Detection

## Technologies Used

- Python
- Deep Learning
- Transformers
- PyTorch / TensorFlow
